## Description

The HDI Prediction System is a web-based tool that predicts a country's Human Development Index (HDI) and classifies it into one of four development tiers — Very High, High, Medium, or Low. Users input key indicators (life expectancy, mean years of schooling, expected years of schooling, and GNI per capita), and the system returns the predicted HDI score along with its development classification, helping users understand how health, education, and income together shape human development outcomes.
